# Chrome版Claudeへの指示：LINE公式アカウントのトークン取得

以下を **Claude for Chrome 拡張機能** にそのまま貼り付けてください。
LINE公式アカウントのMessaging APIチャンネルから「長期アクセストークン」を取得します。

> ゴール：`LINE_CHANNEL_ACCESS_TOKEN` に登録する長期トークンを取得して井上さんに渡すこと。
> これが取得できると、Instagram投稿と同時にLINE公式アカウントにも自動配信されます。

---

## 拡張へのプロンプト（コピペ用）

```
LINE公式アカウントのMessaging APIチャンネルアクセストークンを取得してください。

【1. LINE Developersを開く】
- https://developers.line.biz/console/ を開く
- ログイン済みでなければ、ShinCRAFTの公式LINEに紐づくLINEアカウントでログイン
  （※ログイン認証は井上さん本人の操作が必要な場合があります）

【2. 対象チャンネルを開く】
- プロバイダー一覧から ShinCRAFT のプロバイダーを選択
- 「Messaging API」チャンネルがあれば選択して開く
- もし Messaging API チャンネルが無い場合：
  「新規チャンネル作成」→「Messaging API」を選び、
  チャンネル名「ShinCRAFT」等で作成（公式アカウントと自動連携される）

【3. 長期トークンを発行】
- チャンネルを開いたら上部タブの「Messaging API設定」をクリック
- 下にスクロールし「チャンネルアクセストークン（長期）」の項目を探す
- 「発行」ボタンをクリック（既に発行済みなら表示されているトークンを使う）
- 表示された長いトークン文字列をコピー

【4. 結果を報告】
- コピーしたチャンネルアクセストークンを井上さんに伝える
- もし途中で認証ダイアログやエラーが出たら、その文面を報告
- トークンは秘密情報なので、井上さん本人にのみ伝えること
```

---

## このあと井上さんがすること（1ステップだけ・本人のみ可能）

Chrome版から受け取ったトークンを、**Claude Code Web環境の「環境変数」設定**に登録：

```
LINE_CHANNEL_ACCESS_TOKEN = （受け取った長期トークン）
```

> ⚠️ Claude Code の環境変数変更はセッションからもChrome版からも不可。井上さん本人の操作のみ。

登録後は、次回のInstagram投稿（`/sns`→承認→自動投稿）時に、LINEにも自動で同じ内容が配信されます。
コードは実装・テスト済み（`scripts/lib/line_client.mjs`／`post_to_buffer.mjs`）。

## 注意（正直な制約）

- LINE公式アカウントが **Messaging API 未連携**の場合、手順2でチャンネル作成が必要。
  作成すると公式アカウントが「LINE公式アカウントマネージャー」と「Developers」両方から
  管理できるようになる（既存の友だち・配信には影響しない）。
- ブロードキャストは**全友だちに一斉送信**される。月間無料メッセージ数の上限
  （LINE公式アカウントの料金プラン）に注意。出店告知など重要投稿のみ連動推奨なら、
  その旨を伝えてくれれば連動対象を絞る実装に変更します。
