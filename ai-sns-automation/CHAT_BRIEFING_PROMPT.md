## Claude Codeからチャットにタスクを送る時のフォーマット

Claude Codeが受信箱にチャット向けタスクを書く際のフォーマット:

```
## [YYYY-MM-DD HH:MM] タスクタイトル
[宛先: chat]  [status: pending]  [id: task-...]
**送信元**: (エージェント名)  **優先度**: normal  **種別**: approval_request

### 指示
井上さんへの承認依頼：
[承認してほしい内容の要約]

確認事項：
1. ...
2. ...

### 期待する出力
「承認」または「修正点: ...」を受信箱に追記して返す
```

## チャット経由でChatGPT/Codexに振る場合

コワーク+Chrome版でChatGPT/Codexにアクセスして実行させる。
指示書はClaude Codeが作成 → チャットが確認 → コワークが貼り付け → 結果を受信箱へ。
