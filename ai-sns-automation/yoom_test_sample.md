# Yoomテスト用サンプル：01_投稿待ち→投稿管理

## テスト目的

Google Driveの `01_投稿待ち` に画像を1枚入れたとき、Google Sheetsの `SNS投稿管理台帳_Shincraft` の `投稿管理` シートに1行追加されることを確認する。

## テストに使う固定値

| 項目 | 値 |
|---|---|
| 監視フォルダ | `01_投稿待ち` |
| 監視フォルダID | `17BVeGqN2A7Kj_ppMxGXz-Nejim7UQj0j` |
| 監視フォルダURL | `https://drive.google.com/drive/folders/17BVeGqN2A7Kj_ppMxGXz-Nejim7UQj0j` |
| スプレッドシート | `SNS投稿管理台帳_Shincraft` |
| スプレッドシートID | `1j8R23sZZfF1h7X1X87EyS1f9KxHkYBPr0ZSbRNxK16s` |
| シート | `投稿管理` |

## テスト画像

`01_投稿待ち` に次の名前で画像を1枚アップロードする。

```text
test_shincraft_001.jpg
```

画像の中身は任意でよい。スマートフォン写真でも、テスト用の小さいJPEG/PNGでもよい。

実際の日時とURLはテスト実行時の値に置き換わる。

| 列名 | 入るべき値の例 |
|---|---|
| 管理ID | `SNS-20260525-143012` |
| 登録日 | `2026/05/25 14:30:12` |
| 商品名 | `test_shincraft_001.jpg` |
| 投稿カテゴリ | `商品紹介` |
| 元画像URL | `https://drive.google.com/file/d/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx/view` |
| 生成画像URL | 空欄 |
| Instagram本文 | 空欄 |
| X本文 | 空欄 |
| CTA | 空欄 |
| ハッシュタグ | 空欄 |

| Instagram投稿URL | 空欄 |
| X投稿URL | 空欄 |

## 成功判定

以下をすべて満たせば成功。

| 確認項目 | 成功条件 |
|---|---|
| 行追加 | `投稿管理` シートの最終行に新しい行が1行増えている |
| 管理ID | `SNS-` で始まる値が入っている |
| 登録日 | テストした日時に近い日時が入っている |
| 商品名 | Driveに入れたファイル名と一致している |
| 投稿カテゴリ | `商品紹介` と入っている |
| 元画像URL | DriveのファイルURLが入っている |
| ステータス | `未確認` と入っている |
| 空欄列 | 生成画像URL、本文、CTA、ハッシュタグ、投稿URL、エラー内容などが空欄のまま |

## 失敗判定

以下のどれかに当てはまる場合は失敗として扱う。

| 状態 | 主な確認先 |
|---|---|
| 行が追加されない | YoomのフローボットがONか、Google DriveトリガーのフォルダIDが正しいか |
| 商品名が空欄 | Google Driveトリガーの `ファイル名` アウトプットを選んでいるか |
| 元画像URLが空欄 | Google Driveトリガーの `ファイルURL` または `Webリンク` アウトプットを選んでいるか |
| ステータスが空欄 | Google Sheetsアクションの `ステータス` に固定値 `未確認` を手入力しているか |
| 別シートに入る | シート名が `投稿管理` になっているか |
| 同じテスト行が何行も入る | Yoomのテストを複数回押していないか |

## テスト後の片付け

動作確認だけが目的の場合は、テストで追加された行を確認後に削除してよい。

削除してもよい行の目印:

- 商品名が `test_shincraft_001.jpg`
- ステータスが `未確認`
- 投稿本文や生成画像URLが空欄
