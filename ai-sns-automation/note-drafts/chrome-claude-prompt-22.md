# Chrome版Claude タスク指示書 — note #22 全工程

以下の手順をブラウザ操作で順番に実行してください。
各ステップの結果を記録しながら進めてください。

---

## ステップ1：ChatGPT「note投稿用」チャットで記事本文を生成

1. `https://chatgpt.com` を開く
2. 左サイドバーの「note用」フォルダ内にある **「note投稿用」** チャットを開く
3. 以下のメッセージを送信する：

---
**送信メッセージ（コピーして送信）：**

```
「病院事務の男」シリーズの#22を書いてください。

テーマ：病院事務は、ChatGPT（AIツール）を一度ちゃんと使ってみた方がいい。

【参考：#21のタイトルと要旨】
タイトル：病院事務は、何でも一度拾うくらいの柔軟さがあった方がいい。
要旨：何でも頼まれごとを受け止め、正しい問い合わせ先へつなぐ。担当外でもいったん拾うことで病院全体の流れが見えてくる。ただし便利屋で終わらず、経験を改善力に変えること。

【シリーズの文体ルール】
- 一人称・体験談ベースの語り口
- 短い文を積み重ねるテンポ
- 「ただ、〜」「でも、〜」で場面転換
- 結論は3点まとめで締める
- 敬体（〜です・〜ます）

タイトルは「#22 病院事務は、ChatGPT（AIツール）を一度ちゃんと使ってみた方がいい。」で固定。
本文のみ生成してください。
```

---

4. 生成された本文をすべてコピーして保存する（次のステップで使用）

---

## ステップ2：ChatGPT「noteイラスト用」チャットでイラストを生成

1. 左サイドバーの「note用」フォルダ内にある **「noteイラスト用」** チャットを開く
2. 以下のメッセージを送信する：

---
**送信メッセージ（コピーして送信）：**

```
noteの記事「#22 病院事務は、ChatGPT（AIツール）を一度ちゃんと使ってみた方がいい。」のヘッダーイラストを作成してください。

【イメージ】
- 病院事務の男性がパソコンでChatGPTを使っている場面
- 温かみのあるイラスト風、シンプルで読みやすい
- noteのヘッダー用（横長 1280×670px 程度）
- テキストなし（タイトルは不要）
- 明るく前向きなトーン

シリーズ統一感を保つため、これまでのnoteイラストと同じスタイルで生成してください。
```

---

3. 生成された画像を**ダウンロード**する（ファイル名：`note22_illust.png` など）

---

## ステップ3：Google Drive に画像を保存

1. `https://drive.google.com` を開く
2. 以下のフォルダへ移動する：
   - **AI > ShinCRAFT_SNS自動投稿 > 03_生成画像**
   - フォルダID（直接アクセス）：`https://drive.google.com/drive/folders/1eVUuN8qYuHp7h0h_I13nNS6TwuiHRcT7`
3. ダウンロードした画像をこのフォルダにアップロードする
4. アップロードされたファイルの **共有リンク**（表示用URL）をコピーして保存する

---

## ステップ4：note.com に記事を投稿

1. `https://note.com` を開き、`oshin_2026` アカウントでログイン済みか確認
2. 右上の **「投稿」** ボタン → **「テキスト」** を選択
3. **タイトル欄**に入力：
   ```
   #22 病院事務は、ChatGPT（AIツール）を一度ちゃんと使ってみた方がいい。
   ```
4. **本文欄**にステップ1でコピーした文章を貼り付ける
5. **ヘッダー画像**にステップ2でダウンロードした画像をアップロードする
6. 右上の **「公開設定」** → **「全体に公開」** を選択
7. **「公開する」** ボタンを押して投稿を完了する
8. 公開後に表示される記事URLをコピーして保存する（例：`https://note.com/oshin_2026/n/nXXXXXXXX`）

---

## ステップ5：完了を報告する

以下の内容を Claude Code チャットに貼り付けて報告してください：

```
note投稿完了
URL: （投稿したnoteのURL）
```

この報告を受け取った後、Claude Code が自動でXへのシェアを実行します。

---

## 注意事項

- 各ステップで生成内容がおかしい場合は一度だけ再生成を試みる
- note投稿後のURLは必ず控えておく（X自動シェアに必要）
- ログイン状態が切れている場合は再ログインして続行する
