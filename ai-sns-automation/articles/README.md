- Claude Code でSNS投稿・note→Xクロスポストを **本当に自動化して回している**
- それを **PCではなくAndroidスマホから指示するだけ** で運用している

→ 「AIで稼ぐ系」は飽和気味だが、**実体験＋具体的な再現手順＋テンプレ** がそろうと一気に売り物になる。

## 各記事 共通の型（参考記事を踏襲）

1. **フック（無料）**：自分の実体験・痛みから入る。「私も最初はこうだった」
2. **問題提起（無料）**：読者の悩みを言語化（「発信が続かない」「時間がない」）
3. **解決の全体像（無料）**：何をすれば解決するかをざっくり見せる（ネタバレしすぎない）
4. **〔ここから有料〕具体的な手順**：再現できるレベルの手順・設定・コツ
5. **テンプレ／コピペ素材**：そのまま使える台帳・プロンプト・チェックリスト
6. **失敗例と対処**：自分がつまずいた所（リアルさが価値になる）
7. **アクションプラン**：今日やる3ステップ

## 価格と無料/有料ラインの考え方

- 無料部分は「読者が"自分のことだ"と思って前のめりになる」ところまで。
- 有料ラインは **「具体的な手順・設定値・テンプレが始まる直前」** に置く。
- 初回は安め（500〜780円）で実績（販売数・スキ）を作り、改訂のたびに値上げする。
- noteの「サポート」「マガジン（束売り）」「定期購読」も後から検討。
