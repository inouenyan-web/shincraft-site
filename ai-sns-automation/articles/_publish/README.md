## 貼り方（note）

1. note で「新しい記事を書く」→ タイトルを記事1行目（`#`）からコピー
2. 本文をまるごと貼る
3. **`🔒 ここから先は有料エリアです`** と書いてある区切りの位置で、
   note の「**ここから先は有料エリアです**」ラインを設定する
   （貼り付けた目印テキスト自体は消してOK）
4. 価格を設定して公開

## 出す順番とねらい

- **まず①「06（柱）」を出す。** これが入口。感情で刺さって拡散しやすく、シリーズ全体への導線になる。
- 続けて②③④を出し、各記事末尾の「この人が書いてる他の話」で相互に回遊させる。
- note の **マガジン機能**で4本を1つに束ねると、まとめ買いされやすい。

## 無料連載（oshin_2026）からの誘導

- 無料連載の記事末尾に1行：「※もっと突っ込んだ"辞め時"や"業務改善の全技術"は有料マガジンに書いてます」
- 特に無料連載の人気テーマ（離職・業務改善・辞めたい時）と内容が地続きなので、
  該当する無料記事の最後から、対応する有料記事へリンクすると効く。

## 価格の考え方

- 最初は下限（780〜880円）でスタートし、販売数・スキの実績を作る。
- 反応を見て、改訂のたびに値上げ。まとめ買い用にマガジン価格も検討。
