## 戦略：信頼（権威）は共通、ジャンルだけ変えて有料化

無料で築いた「**本物の病院職員**」という信頼は、実務ノウハウ以外でも効く。
実務How-toとは**別ジャンル**＝「**中の人しか書けない人間ドラマ・恋愛事情の実話エッセイ**」を
有料の主力にする。無料読者が「同じ人の、もっと生々しい話」を読みたくて有料へ流れる導線。

| | 無料連載（集客） | 並行する有料記事（収益） |
|---|---|---|
| ジャンル | 実務ノウハウ・キャリア論 | 実話エッセイ（恋愛・人間模様） |
| 読者の動機 | 学びたい | 中の人しか知らない話を覗きたい |
| 文体 | おしんの関西弁 | そのまま流用（強みになる） |

## 【方向性アップデート】恋愛・修羅場だけでなく「働くリアル全部」を、リクルート向けに

エンタメ寄りの恋愛・修羅場"だけ"より、**「中の人が全部書く、病院で働くリアル」**として
**就職・転職を検討してる人向け（リクルート向け）**に振ると、ターゲットが明確になって売れやすい。
恋愛・修羅場は「リアルの一章」として内包する。

| | 無料連載 | 有料記事（この方向） |
|---|---|---|
| 内容 | 仕事のやり方を上手くする（実務ノウハウ） | 病院で働くリアル全部（入る前に知りたい本当のとこ） |
| 読者 | すでに働いてる事務職 | これから病院で働く人／転職・就活中・医療系学生・異業種から医療事務志望 |
| 価値 | スキルが上がる | 「入って後悔せん」判断材料・覚悟ができる |
| ジャンル | ニッチ実務 | キャリア・転職（財布が開きやすい層） |

フラッグシップ：`oshin_paid_02_real.md`（病院で働くリアル全部／リクルート向け）。
この中に「恋愛・修羅場」「給料」「人間関係」「やりがい」「しんどさ」を章立てで収める。
`oshin_paid_01_renai.md`（恋愛エッセイ）は、そこから派生する"もっと生々しい単品"として残す。

## 無料→有料の導線（具体）

1. 無料連載の記事末尾に1行：「※病院の"裏側の人間ドラマ"は有料マガジンに書いてます」
2. 有料記事の無料部分（冒頭）は、無料連載と同じ関西弁トーンで安心させる
3. 「結論／核心を知りたい！」の頂点で有料ラインを引く
4. 価格は最初500〜780円で実績（販売数・スキ）を作り、改訂ごとに値上げ
5. 数本たまったら note の「マガジン」で束売り or 定期購読も検討

## 守るべきこと（炎上・トラブル回避）

- **個人が特定されない**：実在の同僚・患者・病院名は仮名／ぼかす。複数の事例を合成する。
- 守秘義務・プライバシーに触れる具体（病名と個人の結びつき等）は書かない。
- 「実話ベースのエッセイ（再構成あり）」と明記しておくと安全。
- 事実部分（本人の体験・感情）は本人にしか書けない → 下書きの `【要記入】` を埋める。

## 文体ガイド（おしんトーンの再現）

- 一人称・関西弁まじりの語りかけ（「〜やで」「〜やねん」「正直な」）
- 説教くさくしない。情景→会話→本音、の順で淡々と。
- 1記事1テーマ、結末で少しだけ余韻（教訓を押し付けない）。
