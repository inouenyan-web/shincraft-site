# /フィード投稿生成

引数 `$ARGUMENTS` でテーマを受け取る（例：`テーマ:レジンアクセサリーの失敗しない色選び`）。

## 手順

1. `employees/research/logs/` から指定テーマに関連するバズ投稿を3件取得
2. 取得した投稿の構成パターンを参考に、10枚構成のフィード投稿を作成
3. 各スライドのテキストを **40文字以内** で書く
4. 構成は必ず **タイトル → 共感 → ノウハウ6枚 → まとめ → CTA** の型を使う
5. `employees/creative/CLAUDE.md` のトーンルールに従う（会話調・共感表現・煽りNG）
6. キャプションを作成（1行目フック、本文箇条書き、CTA1つ、ハッシュタグ大中小ミックス）
7. `employees/marketing/line_sequence/hashtag_list.md` からハッシュタグを選ぶ（なければ新たに提案）
8. 結果を `employees/creative/logs/YYYY-MM-DD_feed_{テーマ}.md` に保存
9. ビジュアル制作指示（色・素材感・レイアウト）を末尾に添える

## 出力先ファイル構造

```markdown
# フィード投稿：{テーマ}（YYYY-MM-DD）

## スライド構成
### S1 タイトル
### S2 共感
### S3〜S8 ノウハウ
### S9 まとめ
### S10 CTA

## キャプション
## ビジュアル指示
```
