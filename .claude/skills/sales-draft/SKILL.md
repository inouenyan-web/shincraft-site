# /sales-draft — 営業・マーケレーン（営業文・提案ドラフト）

## このスキルの担当範囲
- **担当**: 営業文・提案書・DM文・見込み客リスト・訴求コピーのドラフト
- **隣接スキルへ委ねるもの**: Instagram投稿実行→`/sns`、フィード構成→`/フィード投稿生成`、競合調査→`/競合分析`

## リサーチが必要な場合
市場規模・競合・価格感の調査が必要（独立ソース3件超）なら **リサーチ** サブエージェントを呼ぶ。

## トリガー別の手順

### A. 営業文・提案書
1. 送付先（BtoB/BtoC）・商品/サービス・訴求ポイントを確認
2. 構成：**掴み（Pain/共感）→ 提案（解決策）→ 実績/根拠 → CTA（次の行動）**
3. 文体：相手目線・押し売り感なし・1CTA

### B. DM・メッセージ文
1. 関係性（初回/既存フォロワー/リスト）と目的（面談打診/商品案内/LINE誘導）を確認
2. 60〜150文字を目安。相手の投稿・状況への具体的な言及を必ず入れる
3. 初手で営業・値段を出さない

### C. 見込み客リスト・アプローチ設計
1. ターゲット条件を確認（業種・規模・地域・フォロワー数等）
2. スクリーニング基準と優先順位の整理
3. アプローチシーケンス（接触→関係→提案→クロージング）の設計

### D. 結果フォーマット
```
## 営業ドラフト：{案件名}

### ドラフト本文
{本文}

### 訴求ポイント補足
- {差別化・根拠}

### 次のアクション
- [ ] {送付先・タイミング・確認事項}
```

## 対外送信の前に
メール・DM・提案書の送付は **必ず本人確認**。秘書は独断で送らない。
