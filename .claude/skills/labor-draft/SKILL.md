# /labor-draft — 労務・行政レーン（就業規則 / 補助金・許認可ドラフト）

## 士業の線引き（厳守）
- 出力範囲：**論点整理・要件調査・書類ドラフト・「何を社労士/行政書士に相談すべきか」の特定**まで。
- **最終申請・届出・監督署提出・認可は有資格者または本人が行う**。必ず明示する。

## トリガー別の手順

### A. 就業規則・労務書類
1. 従業員数・雇用形態・対象範囲を確認（10人以上なら届出義務あり）
2. 該当条項（労働時間・休暇・懲戒・給与計算など）のドラフト
3. 直近の法改正（育休・同一賃金・36協定など）との整合チェックリストを添付
4. 社労士への相談リストを出す

### B. 補助金・助成金ドラフト
1. 公募要領の読み込みが必要 → **リサーチ** サブエージェントを呼ぶ
2. 要件確認：①申請主体 ②対象経費 ③補助率・上限 ④申請期限 ⑤加点項目
3. 申請書骨子ドラフト（事業概要・導入目的・実施計画・費用積算）
4. 証拠書類チェックリスト（見積書・決算書・登記簿等）

### C. 許認可・行政手続き
1. 業種・許認可の種類を特定（建設業許可・古物商・食品営業等）
2. **リサーチ** で都道府県ごとの要件・窓口・提出書類を調査
3. 申請書記載事項のドラフト ＋ 添付書類リスト

### D. 全案件共通の結果フォーマット
```
## 労務・行政ドラフト：{案件名}

### 書類ドラフト / 要件整理
{本文}

### 有資格者への相談リスト
1. {要相談点}

### 次のアクション
- [ ] {具体的な手順（期限付き）}
```

## 不可逆操作の前に
申請提出・署名・費用支払いは **必ず本人承認**。秘書は独断で実行しない。
