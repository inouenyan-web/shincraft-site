# /コメント文一括生成

引数 `$ARGUMENTS` で件数を受け取る（例：`件数:20`）。未指定なら20件。

## 手順

1. `employees/research/logs/daily/` の最新ファイルから、ターゲットアカウントの今日の投稿を抽出
2. 各投稿について `employees/community/CLAUDE.md` のルールに従ってコメント文を作成：
   - 60〜100文字
   - 投稿内容に具体的に言及
   - 自分の経験・感想を1〜2文添える
   - 質問で終わらせる
   - 「素敵ですね」だけの定型文NG
3. 指定件数分を生成
4. 結果を `employees/community/comments/YYYY-MM-DD.md` に保存（送信先アカウントリスト付き）

## 出力フォーマット

```markdown
# YYYY-MM-DD コメント文案

件数：X件

---

### 1. @{account_name}
投稿テーマ：{投稿内容の概要}

コメント文：
{コメント本文}

---
```
