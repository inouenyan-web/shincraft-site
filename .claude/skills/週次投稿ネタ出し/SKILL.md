# /週次投稿ネタ出し

来週分のインスタ投稿ネタを5案まとめて出す。

## 手順

1. `employees/research/logs/` から直近1週間の日次・週次レポートを全て読む
2. `employees/secretary/ideas/` から最近のネタ候補メモを取得する
3. 上記を組み合わせて来週分の投稿ネタを5案出す
4. 各ネタについて「フィード投稿向き / リール向き / ストーリーズ向き」を判定
5. 各ネタに「想定タイトル」と「狙いの数字（保存数・コメント数等の目標）」を付ける
6. 結果を `employees/creative/logs/YYYY-Www_next_week_ideas.md` に保存（YYYY-Www は今週の週番号）
7. `employees/secretary/todos/YYYY-MM-DD.md` に「来週ネタ確認待ち」を追加

## 出力フォーマット

```markdown
# YYYY-Www 来週投稿ネタ案

生成日：YYYY-MM-DD

| No | テーマ | 投稿種別 | 想定タイトル | 目標KPI |
|----|--------|---------|------------|--------|
| 1  |        |         |            |        |
| 2  |        |         |            |        |
| 3  |        |         |            |        |
| 4  |        |         |            |        |
| 5  |        |         |            |        |

## 各ネタの詳細メモ
### ネタ1：
### ネタ2：
...
```
