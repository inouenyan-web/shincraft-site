# /legal-draft — 法務・紛争レーン（書類ドラフト／交渉準備）

## 士業の線引き（厳守）
- 出力範囲：**論点抽出・書類ドラフト・交渉戦略の整理・「何を弁護士に相談すべきか」の特定**まで。
- **最終判断・提出・係争・登記・法的効力を問われる場面は弁護士へ**。必ず明示する。

## トリガー別の手順

### A. 契約書・規約ドラフト
1. 目的と当事者（甲乙）・主な条件を確認（曖昧な場合だけ1問）
2. 該当する雛形構成（業務委託 / 秘密保持 / 売買 / 利用規約 等）を提示して選択
3. 条項ドラフト作成（重要条項：目的・対価・期間・解除・損害賠償・管轄）
4. **専門レビュー** を呼び、不利条項・抜け・リスク配分を点検

### B. 内容証明ドラフト
1. 差出人・相手方・請求内容（金額・期限）・経緯を確認
2. 内容証明の要件（字数・行数・記載事項）を踏まえてドラフト
3. 送付方法（書留・電子内容証明等）の選択肢を案内

### C. 交渉・調停準備
1. 争点と自分側の主張・相手側の主張を整理
2. **中立審査** を呼び、相手の最強反論を想定して弱点を補強
3. 交渉スクリプト（冒頭・主張・落とし所・退き際）のドラフト作成
4. 調停申立の場合：申立書の骨子ドラフト ＋ 準備すべき証拠リスト

### D. 全案件共通の結果フォーマット
```
## 法務ドラフト：{案件名}

### 書類 / 戦略ドラフト
{本文}

### 弁護士への相談リスト
1. {要相談点}

### 次のアクション
- [ ] {具体的な手順}
```

## 不可逆・対外送信の前に
契約締結・内容証明の発送・調停申立は **必ず本人承認**。秘書は独断で実行しない。
