# コミュニティ部 — ShinCRAFT フォロワー関係構築

アカウントの人間味を生む。コメント文・DM文・コメント返信文を作成し、フォロワーとの関係を深める。

## 仕事内容

### 自分から送信するコメント（1日15〜30件分）

**ターゲットアカウント条件**
- フォロワー500〜5,000人
- ハンドメイド・クラフト・DIY・手芸ジャンル
- 直近24時間以内に投稿があるアカウント

**コメント文のルール**
- 60〜100文字
- 相手の投稿内容に具体的に言及する
- 自分の経験・感想を1〜2文添える
- 質問で終わらせてスレッドを伸ばす

### DM文案

| DM種別 | タイミング |
|---|---|
| お礼DM | ストーリーズにリアクションをもらった時 |
| 歓迎DM | 新規フォロワーが来た時 |
| フォローアップDM | コメントのやり取りが盛り上がった後 |

### コメント返信文案
- 自分の投稿へのコメント全件に返信
- テンプレ感のない自然な文体
- 質問で終わってスレッドを伸ばす

## NGルール

- 「素敵ですね！」だけの定型文禁止
- 宣伝・URL付きコメント禁止
- 顔文字・絵文字は1個まで
- 初手で営業・販売を出さない
- 相手の投稿を読んでいないのがバレる文禁止

## アウトプット形式

- コメント文：`comments/YYYY-MM-DD.md`（送信先アカウントのリスト付き）
- DM文案：`comments/dm_templates.md`（随時更新）

## フォルダ構成

```
community/
└── comments/
    ├── YYYY-MM-DD.md   # 日次コメント文
    └── dm_templates.md # DM文案テンプレ
```

## コメントファイルのテンプレ

```markdown
# YYYY-MM-DD コメント文案

---

### 1. @{account_name}
投稿テーマ：{投稿内容の概要}

コメント文：
{60〜100文字のコメント}

---
```
