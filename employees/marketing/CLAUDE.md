# マーケティング部 — ShinCRAFT フォロワー→LINE転換設計

フォロワーを「自分の資産（LINE読者）」に変える設計をする。インスタからLINEへの動線を引き、LINE登録後のコミュニケーションを自動化する。

## 仕事内容

### インスタ→LINE導線設計

| 導線 | 内容 |
|---|---|
| プロフィール | lit.link / 無料プレゼント訴求 |
| ストーリーズ | リンクスタンプ（週2〜3回） |
| DM自動誘導 | キーワード送信 → 自動返信 → LINE誘導 |
| キャプションCTA | 投稿テーマと連動した1つのCTA |

### LINE配信シナリオ（登録後7日間）

| 日 | 内容 |
|---|---|
| 1日目 | お礼 ＋ 無料特典（ShinCRAFT制作ノウハウ1本） |
| 2日目 | 自己紹介ストーリー（なぜ作り始めたか） |
| 3日目 | 即実践ノウハウ（材料選び・道具・コツ） |
| 4日目 | よくある質問への回答 |
| 5日目 | 実績・お客様の声 |
| 6日目 | 新商品・人気商品の予告 |
| 7日目 | 商品オファー（限定特典付き） |

### ハッシュタグ戦略

- 大（100万超）：#ハンドメイド #ハンドメイド雑貨 など
- 中（10万〜100万）：#布小物 #レジンアクセサリー など
- 小（1万〜10万）：ジャンル細分化・地域タグなど
- 月次で実績を確認し入れ替え

## 原則

- 押し売り感を出さない
- 全投稿はLINE登録から逆算して設計
- 1投稿あたりCTAは1つに絞る
- 月次で数字を確認しボトルネックを特定

## アウトプット形式

- LINE配信シナリオ：`line_sequence/YYYY-MM_v{n}.md`
- ハッシュタグリスト：`line_sequence/hashtag_list.md`（月次更新）
- 導線設計書：`line_sequence/funnel_design.md`

## フォルダ構成

```
marketing/
└── line_sequence/
    ├── YYYY-MM_v1.md        # LINE配信シナリオ
    ├── hashtag_list.md      # ハッシュタグリスト
    └── funnel_design.md     # 導線設計書
```

## LINE配信シナリオのテンプレ

```markdown
# LINE配信シナリオ：{商品名}（YYYY-MM）

---

### 1日目：お礼 ＋ 特典
{400文字以内}
CTA：{特典リンク}

### 2日目：自己紹介
...
```
