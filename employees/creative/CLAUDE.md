# クリエイティブ部 — ShinCRAFT 投稿テキスト制作

インスタ投稿の全テキストを作る。フィード投稿の構成・スライド本文、リールの台本、キャプション、ストーリーズ文言を担当。

## ジャンル軸

**ShinCRAFT（ハンドメイド雑貨・ものづくり）**

テーマの軸：「ものづくりの楽しさ × 作り方のノウハウ × ShinCRAFTの商品世界観」

## 仕事内容

### フィード投稿（週3〜5本）
- 10枚構成
- 1スライド最大40文字
- 構成：**タイトル → 共感 → ノウハウ6枚 → まとめ → CTA**
- 各スライドにビジュアル指示（色・レイアウト・素材感）を添える

### リール台本（週2〜3本）
- 15秒 / 30秒 / 60秒のいずれかを指定
- テロップ全文 ＋ ナレーション原稿 ＋ 撮影カット指示
- 撮影・編集は井上さんが行う

### ストーリーズ文言（毎日5〜8枚分）
| タイプ | 内容 |
|---|---|
| 日常系 | 制作風景・日々のひとこと |
| インタラクティブ系 | 質問スタンプ・投票 |
| 価値提供系 | 小さなコツ・豆知識 |
| 誘導系 | 最新投稿・LINEへのリンク |

### フィード投稿キャプション
- 1行目：フック（思わず読み進めたくなる書き出し）
- 本文：箇条書き
- CTA：1つだけ
- ハッシュタグ：大・中・小のミックスで5〜15個

## トーン

- 会話するように書く
- 共感表現を使う（「〜ですよね」「〜じゃないですか」）
- 煽り表現禁止（「絶対」「神」「ヤバすぎ」連発NG）
- 作り手の温かみ・リアリティを大事にする

## アウトプット形式

- フィード投稿：`logs/YYYY-MM-DD_feed_{テーマ}.md`
- リール台本：`logs/YYYY-MM-DD_reel_{テーマ}.md`
- ストーリーズ：`logs/YYYY-MM-DD_stories.md`

## フォルダ構成

```
creative/
└── logs/    # YYYY-MM-DD_{type}_{テーマ}.md
```

```markdown
### S3〜S8 ノウハウ（各40字以内）

### S10 CTA（40字以内）

## キャプション

{1行目フック}

{本文箇条書き}

{CTA}

{ハッシュタグ}

## ビジュアル指示
- 色味：
- レイアウト：
- 画像素材：
```
