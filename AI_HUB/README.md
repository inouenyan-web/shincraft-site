# AI Collaboration Hub

## What this adds
This folder is the lightweight quality-control layer shared by Claude, ChatGPT, Codex, and the owner. It complements existing project assets; it does not replace them.

## Existing source-of-truth assets
- `CLAUDE.md`: repository-wide operating rules for Claude Code.
- `LESSONS.md`: confirmed lessons and user corrections shared by all AI tools.
- `AI共有ブリーフィング_最新.md`: latest operations briefing generated by existing automation.
- `運用ボード.md`: active operational work.
- `ai-sns-automation/`: implementation, scripts, and product-specific rules.

## This hub's job
- Preserve a concise, public-safe history of confirmed work.
- Separate current priorities from permanent rules.
- Run an independent daily quality review so one AI can catch omissions, contradictions, or overly narrow plans from another.
- Turn only high-value findings into GitHub Issues or pull requests.

## Files
- `PROJECT_HISTORY.md`: durable non-sensitive history.
- `CURRENT_STATE.md`: current work, blockers, and next actions.
- `DECISIONS.md`: approved decisions that affect future work.
- `DAILY_REVIEW_PROMPT.md`: fixed review protocol for Claude and ChatGPT.
- `daily-reviews/`: dated outputs. Create only when there is a material finding.

## Rule of thumb
`LESSONS.md` is the authoritative learning log. This hub summarizes and operationalizes it; it must not contradict it.
